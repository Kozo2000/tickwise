# 投資家・トレーダー向け活用ガイド (Investor Guide)

Tickwiseは、投資家が市場の「ノイズ」に惑わされず、客観的な事実に基づいて冷静な判断を下すための支援ツールです。本ガイドでは、投資分析におけるツールの位置付けと活用方法を解説します。

---

## 1. 株式投資分析におけるTickwiseの役割

株式投資の分析には、大きく分けて「ファンダメンタル分析」と「テクニカル分析」の2つがあります。

- **ファンダメンタル分析**: 企業の財務・業績やニュース（材料）から「何を買うべきか（価値）」を判断します。
- **テクニカル分析**: 過去の株価や出来高のパターンから「いつ買うべきか（タイミング）」を判断します。

Tickwiseは、この**「テクニカル分析（客観的な数値）」と「最新材料（ニュース）」をAI（LLM）の脳を使って統合**し、あなたに「今の市場の体温」を解釈可能なレポートとして提供します。

> 各テクニカル指標の具体的な計算方法や見方は、[**インジケーターガイド**](./indicator-guide.md) を参照してください。

## 2. ツールとしての特性と推奨されるプレイスタイル

### テキストベースへの特化と「共存」の思想
Tickwiseにはグラフ表示機能がありません。これは単なる開発上の制約ではなく、**「既存の優れたチャートサービスと競うのではなく、それらを補完する」**という設計思想に基づいています。

TradingViewや各証券会社の高度なチャート表示機能は非常に素晴らしく、Tickwise単体でそれらに対抗することは不可能です。だからこそ、Tickwiseは「目に見えるチャート」を補完する**「目に見えないデータの裏付け」を言語化すること**に特化しています。

1.  **チャートサービス**: 視覚的に全体のトレンド、抵抗線、パターンを確認する。
2.  **Tickwise**: その時、内部的な数値（RSI, MACD, ニュースの文脈等）がどう連動しているかを論理的に確認する。

このように、優れた視覚ツールと「共存」してお使いいただくことで、より精度の高い投資判断が可能になると確信しています。他のサービスと共にお使いいただくことで、分析にさらなる深みと客観性を加え、あなたの判断プロセスをそっと支える「副官」のような存在を目指しています。

### パラメータによる銘柄性格の最適化
投資戦略は、短期トレード、スイング、長期投資など人それぞれです。Tickwiseは多数の引数により、分析の「厳しさ」や「視点」を自由に変更できます。
- **買いたい時**: `-s buyer`（買い手視点）で分析。
- **売りたい時**: `-s seller`（売り手視点）で警戒ポイントを確認。

> 分析戦略のカスタマイズについては [**ストラテジーガイド**](./strategy_Guide.md) を参照してください。

## 3. 対応市場について

Tickwiseは、基本的に **Yahoo Finance API** がデータを提供している市場であれば、グローバルに対応可能です。

- **日本市場**: ティッカー末尾に `.T` を付与（例: `7203.T`）
- **米国市場**: ティッカーをそのまま入力（例: `AAPL`, `NVDA`）
- **その他市場**: Yahoo Financeで検索可能なティッカーであれば、ロジックは共通して適用されます。
