## 🎯 プロダクトの独自性と提供価値

Tickwiseは、市場にある多くの分析ツールとは一線を画し、「**ユーザーが分析の主導権を握る自由なUX**」と「**AIとの論理的な協調**」を軸に設計されています。

### 1. ユーザー主体でカスタマイズ可能なUX
既存の多くのツールは計算ロジックがブラックボックス化されていたり、設定が固定されていたりすることが少なくありません。
Tickwiseは、RSIの閾値、移動平均の期間、各指標がスコアに与える影響度（重み付け）など、数十種類のパラメータを1つのコマンド引数で自由自在にチューニング可能です。ツールに決められた評価を押し付けられるのではなく、あなたの投資哲学をツールに投影するためのインターフェイスを提供します。

### 2. 「事実」と「推論」を分離し、AIの真価を引き出す
AIに価格予測を丸投げするのではなく、Rustで厳密に計算された「確定データ」をAIに供給することに特化しています。
これにより、AIはあやふやな計算作業から解放され、客観的データに基づいた高度な文脈理解と判断支援のみに従事できるため、ハルシネーション（もっともらしい嘘）を物理的に抑制した分析を実現します。

### 3. テクニカルとニュースの「多角的融合」
投資判断において、数値（テクニカル）と材料（ニュース）は車の両輪です。Tickwiseは、Brave Search API等を利用した最新ニュースの自動収集・サマライズ機能を搭載しています。
「算出された客観的指標」と「市場を動かしている最新の話題」をAIが一つのコンテキストとして統合して読み解くことで、単一のデータソースでは到達できない多角的な分析支援を提供します。

### 4. 日米市場への最適化
東京証券取引所（TSE）とニューヨーク証券取引所（NYSE）を自動識別し、各市場の運用ルールや時間軸に基づいた整合性の高い解析をシームレスに行います。

## ✨ 主な機能

- **多角的なテクニカル解析**: RSI, MACD, ボリンジャーバンド, 一目均衡表, フィボナッチなどの主要指標から、ADXやROCなどの拡張指標まで幅広く対応。
- **ニュース・サマライズ機能**: 指定したティッカーに関連する最新のニュースを自動取得し、LLMが重要度を見極めて要約。
- **テクニカル×ニュース統合分析**: 指標データとニュース内容を織り交ぜたプロンプトを自動生成し、AIによる高度な投資判断支援レポートを作成。
- **LLM連携エンジン**: 推論の補助脳としてOpenAI等のLLMを利用。分析結果を構造的なレポートとして出力。

## 🛡 設計の信頼性

本ツールは、資産管理を補助する道具としての信頼性を確保するため、以下の設計方針で実装されています。

- **メモリ安全性と堅牢性**: Rust言語を採用し、パニック（異常終了）を極力排除したエラーハンドリングを徹底しています。
- **機密保護**: APIキーなどの機密情報はメモリ上の滞留時間を極小化し、利用直後にゼロパディング（0クリア）を行っています。
- **アルゴリズムの透明性と独立性**: 指標計算には一般に公表されている数学的アルゴリズムおよび標準的なオープンソースライブラリ（`ta` crate等）を採用し、独自に実装されています。特定の商用製品の模倣を目的としたものではなく、先端技術の統合を追求する独立した実験的プロジェクトです。
- **データの一貫性 (SOT)**: 指標算出後のデータは内部的にカプセル化（`TechnicalDataGuard`）され、レポート生成過程での数値の等価性を保証しています。
